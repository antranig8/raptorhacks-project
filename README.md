# RaptorHacks Project

## Overview
This project is a full-stack application built for RaptorHacks. It uses a React frontend, a Flask backend for API and machine learning functionality, and Supabase for the database.

---

## Tech Stack
- **Frontend:** React  
- **Backend:** Flask (Python)  
- **Database:** Supabase  
- **Deployment:**
